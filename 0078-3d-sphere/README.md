## STARTING OFF:

Make sure you have npm installed.
Run:
```
npm install http-server -g
```

Run:
```
http-server
```

View at: http://localhost:8080/0078-3d-sphere

* NOTE: moved vendor and images out of folder.
  * Running from main directory instead inside this directory.

* Drawing a 3d Sphere
